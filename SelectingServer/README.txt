The server can be run with './Server'. 
Takes a port number as an argument. If no argument is given it will use deafult port number 48040.

The client can be run with './Client 127.0.0.1'.
Requires an address to connect to as an argument.
Takes a port number as a second argument. If no argument is given it will use deafult port number 48040.
Untested outside use of local machine.
